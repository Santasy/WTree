La presente carpeta cuenta con:
1. El archivo `SingleOperations.cpp`, que permite ejecuciones automáticas de todas las operaciones según la cantidad especificada en argumentos, además de una intervención manual por etapas.
1. La carpeta `UnitTests`, que cuenta con el archivo `TestNamespace` donde se definen diversas funciones con dos enfoques:
    - demostrar comportamientos de borde.
    - realizar benchmarks para un gran número de repeticiones.
1. Las carpetas finalizadas con "_Test" representan códigos que se encargan de validar aspectos específicos de la estructura.
    > Para realizar las 3 pruebas se puede ejecutar `make`
